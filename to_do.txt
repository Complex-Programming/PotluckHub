Phase 6+
The Frontend Button: James will delete the <select> dropdown and install @react-oauth/google. He will drop a <GoogleLogin /> button onto the header.

The Handshake: When a user clicks that button and logs in with their Gmail, Google sends James a massive string of gibberish called a JWT (JSON Web Token).

The Backend Verification: James sends that JWT to Tom's API (POST /api/auth/google). Tom uses Google's official Node.js library to decode it. This guarantees the user isn't faking their identity.

The Database Sync: Tom looks at the decoded email (e.g., james@gmail.com).

If that email is already in the user table, Tom logs them in.

If they are brand new, Tom runs INSERT INTO user (name, email) VALUES ... to add them.
